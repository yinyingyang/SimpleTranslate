# SimpleTranslate

## 简介

一个用于ComfyUI的简单翻译节点插件，可以根据输入的整数选择对应的文本，并支持翻译开关控制。
在使用comfyUI时，经常需要调整提示词来满足不同的需求，而这个节点可以帮助您快速切换不同的提示词。
部分模型（如flux）不支持中文提示词，而这个节点可以帮助您在中文和英文之间切换提示词，且提供注释功能方便更多操作（如网上找到的英文提示词效果会比翻译的更好，但一大段英语我们自己又看不懂，就可以用注释来说明）。

## 功能

- 根据输入的整数（1-5）选择对应的文本
- 支持最多5个不同的文本输入
- 可选择是否翻译选中的文本
- 支持多行文本输入
- 以'#'开头的行会被视为注释，运行时会被忽略

## 安装

1. 将此文件夹复制到ComfyUI的`custom_nodes`目录下
2. 重启ComfyUI
3. 在节点菜单中找到"SimpleTranslate"分类

## 使用方法

1. 在节点工作区中添加"简单翻译"节点
2. 输入1-5的数字来选择对应的文本
3. 在文本框中输入您的内容（支持多行）
4. 使用"翻译"开关控制是否翻译选中的文本
5. 节点将输出处理后的文本

## 依赖

- argostranslate：用于文本翻译功能，秋叶大佬的整合包已包含，无需额外安装

## 注意事项

- 翻译功能默认将中文翻译为英文
- 以'#'开头的行将被视为注释并自动删除
- 索引超出范围时将自动调整到最近的有效值
